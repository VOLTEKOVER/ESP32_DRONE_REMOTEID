📦 Arduino IDE

                    

                        

                            
1

                            

                                Apri il file .ino
                                
Seleziona il file desiderato dal progetto

                            

                        

                        

                            
2

                            

                                Esporta il binario
                                
Sketch → Export Compiled Binary

                            

                        

                        

                            
3

                            

                                Recupera il file

⚡ PlatformIO

                    

                        

                            
1

                            

                                Compila il progetto
                                
Esegui il comando build

                            

                        

                        

                            
2

                            

                                Attendi il completamento
                                
Il processo richiede qualche minuto
